# Tojemoje — web page

Single-page static website (PL/EN) created in this folder.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Run locally
From this folder:

```powershell
python -m http.server 5173
```

Open:
- http://localhost:5173/

If Python isn’t available, you can use any static server.
